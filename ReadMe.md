# Spring Core Learning Repository

> A structured repository documenting my journey of learning **Spring Core fundamentals (before Spring Boot)**.

---

## 📌 About

This repository is a collection of **hands-on notes, examples, and experiments** while learning the **Spring Framework Core concepts**.

The goal is to:
- Build strong fundamentals of Spring
- Understand internal working before using Spring Boot
- Practice concepts through small code snippets

---

## 🎯 What I'm Learning

- Spring IoC Container
- Dependency Injection (DI)
- Bean Management
- ApplicationContext vs BeanFactory
- XML & Annotation-based configuration
- Bean Lifecycle
- Loose Coupling principles

---

## 🧠 Topics Covered

### 🔹 Core Concepts
- Inversion of Control (IoC)
- Dependency Injection
    - Constructor Injection
    - Setter Injection
- Spring Beans
- Bean Scopes (Singleton, Prototype)

### 🔹 Configuration
- XML Configuration
- Annotation-based Configuration
    - `@Component`
    - `@Autowired`
    - `@Configuration`

### 🔹 Internals
- Bean Lifecycle
- Autowiring
- Component Scanning
